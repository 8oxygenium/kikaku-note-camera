記録メディア

連写、動画ビットレート、冗長記録に影響

規格特徴主な用途注意点
SD / SDHC / SDXC最も普及したカード規格。UHS-I、UHS-II、V30、V60、V90 などがある。静止画、一般的な4K動画、日常撮影同じSDでも速度差が大きい。動画では最低保証速度を確認。
CFexpress Type A小型で高速。対応機種は一部ミラーレスに限られる。高ビットレート動画、長時間連写SDより高価。スロット互換とカードリーダーを確認。
CFexpress Type B大型でさらに高速なプロ向けメディア。8K、RAW動画、超高速連写発熱、容量単価、対応スロットを確認。
microSD小型機器で多いカード規格。アクションカメラ、ドローン、小型収録機アダプター運用時の相性と耐久性に注意。

動画記録形式

編集耐性、容量、再生互換性に影響

項目例特徴確認ポイント
解像度FHD / 4K / 6K / 8K高解像度ほどトリミング耐性が上がり、データ量も増える。クロップ、全画素読み出し、オーバーサンプリング。
フレームレート24p / 30p / 60p / 120p60p以上は滑らかな動きやスロー再生に向く。音声記録、AF、時間制限、熱停止。
色深度8bit / 10bit10bitは色編集時の破綻を抑えやすい。内部記録か外部記録か、Log対応の有無。
圧縮方式H.264 / H.265 / ProResH.265は高効率、ProResは編集しやすい。編集PCの再生負荷とカード速度。

手ブレ補正

静止画と手持ち動画の安定性に影響

方式略称例特徴向く場面
ボディ内補正IBIS撮像素子を動かして補正。非対応レンズでも効く場合がある。単焦点、広角、手持ち撮影
レンズ内補正OIS / IS / VR / OSSレンズ側で光学的に補正。望遠域で有利なことが多い。望遠、スポーツ、野鳥
協調補正Dual I.S. / Sync ISボディとレンズの補正を組み合わせる。低速シャッター、手持ち動画
電子補正電子IS / Active映像をクロップして揺れを抑える。歩き撮り、Vlog

AF方式

ピント合わせの速度、追従性、動画安定性に影響

方式特徴強み注意点
像面位相差AF撮像面上の位相差画素で距離方向を判断する。追従AF、動画AF、動体撮影対応エリアや低照度性能は機種差が大きい。
コントラストAFコントラストが最大になる位置を探す。静物で高精度になりやすい。前後に迷うウォブリングが出る場合がある。
ハイブリッドAF位相差AFとコントラストAFを組み合わせる。速度と精度の両立。レンズや撮影モードで性能が変わる。
被写体認識AF人物、動物、鳥、車両などを検出して追従する。構図を保ったまま撮影しやすい。認識対象と照明条件を確認。

端子規格

給電、外部収録、音声、同期まわりを確認

端子用途特徴確認ポイント
USB-C充電、給電、データ転送、Webカメラ近年の標準端子。USB PD、転送速度、UVC/UAC対応。
HDMI外部モニター、外部レコーダーType A、Type C、Type D などサイズ差がある。クリーン出力、10bit、RAW出力、端子強度。
3.5mmマイク外部マイク入力音質改善に使う基本端子。プラグインパワー、入力レベル調整。
3.5mmヘッドホン録音モニター収録中のノイズやレベルを確認できる。動画用途では有無を確認。

初心者向け用語

仕様表で見かける基本語

用語意味見るポイント
換算焦点距離フルサイズ基準に置き換えた画角の目安。APS-Cやマイクロフォーサーズで望遠寄りに見える。
F値レンズの明るさとボケ量に関係する数値。小さいほど明るく、背景をぼかしやすい。
クロップセンサーの一部を切り出して記録すること。動画時の画角変化に注意。
RAW画像処理前の情報を多く残す記録形式。編集耐性は高いが容量が大きい。
Log広い階調を残す動画撮影用プロファイル。色編集前提。露出管理とLUT運用が必要。
